# Hostel Management System

## Description
A role-based web application developed to manage hostel operations such as room allocation, fee tracking, attendance, and complaint handling.

## Features
- Role-based access (Admin, Warden, Student)
- Room allocation and fee management
- Attendance tracking system
- Complaint management system
- Dashboard with real-time updates

## Technologies Used
- HTML
- CSS
- JavaScript
